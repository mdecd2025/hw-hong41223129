詳細說明如何手動方式在 Webots 建立下列場景

要手動在 Webots 中建立場景，您可以按照以下步驟進行：

  1. 啟動 Webots：開啟 Webots 軟體。
  2. 創建新世界 (New World)：選擇 File -> New World，創建一個新的仿真世界。
  3. 設置場景元素：在左側的“場景樹”中，您可以選擇添加不同的物體，如機器人、地面、障礙物等。選擇“Add”按鈕來添加物體，並選擇所需的物件。
  4. 配置機器人模型：選擇一個機器人模型，並配置其所需的參數。
  5. 設置傳感器：為您的機器人添加各種傳感器，如距離感測器、相機等。
  6. 保存並測試場景：設定完成後，保存場景並運行仿真來測試您的場景。

一、繁體中文詳細說明

Webots 是一款由 Cyberbotics Ltd. 開發的開源 3D 機器人模擬器，廣泛應用於工業、教育和研究領域。

Webots 的開發始於 1996 年，最初由瑞士洛桑聯邦理工學院（EPFL）的 Olivier Michel 博士開發，後來在 1998 年由 Cyberbotics Ltd. 作為專有軟體進行開發。

Webots 提供大量可自由修改的機器人、感測器、致動器和物體模型。使用者可以從頭開始構建新模型，或從 3D CAD 軟體中導入模型。在設計機器人模型時，使用者需要指定物體的圖形屬性（如形狀、尺寸、位置、方向、顏色和材質）和物理屬性（如質量、摩擦係數、彈簧和阻尼常數）。軟體中還包含簡單的流體動力學模擬功能。

Webots 使用 ODE（Open Dynamics Engine）來檢測碰撞並模擬剛體動力學。ODE 函式庫允許精確模擬物體的物理屬性，如速度、慣性和摩擦力。

It supports multiple programming languages, including C, C++, Python, Java, and MATLAB. Webots includes a large collection of freely modifiable models of robots, sensors, actuators, and objects, enabling users to build new models from scratch or import them from 3D CAD software. 四、手動建立場景的詳細步驟
假設您希望在 Webots 中建立一個簡單場景：一個機器人在平面上移動並避開障礙物。以下是手把手說明如何透過圖形介面完成：
步驟 1：啟動 Webots
  1. 打開 Webots 應用程式。
  2. 在啟動畫面選擇「New World」（新建世界）。
步驟 2：設置基本場景
  1. 加入地板
    • 在頂部選單點擊「Add」（添加）。
    • 選擇「Solid」（實體物件） > 「Box」（方塊）。
    • 在屬性面板中設置：
      • 大小（Size）：10m x 10m x 0.1m（寬廣的平面）。
      • 名稱（Name）：Floor。
    • 注意：確保地板位於 Z=0，避免懸浮。
  2. 加入機器人
    • 點擊「Add」 > 「Robot」（機器人）。
    • 在下拉選單中選擇「E-puck」（一個簡單的雙輪機器人）。
    • 調整位置（Translation）：例如 (0, 0, 0.05)，使其位於地板上。
    • 注意：檢查機器人是否與地板接觸，避免穿透或漂浮。
  3. 加入障礙物
    • 點擊「Add」 > 「Solid」 > 「Box」。
    • 設置大小：1m x 1m x 0.5m。
    • 位置（Translation）：(2, 0, 0.25)。
    • 名稱：Obstacle。
步驟 3：添加控制器
  1. 點擊機器人節點（E-puck）。
  2. 在屬性面板中找到「Controller」欄位。
  3. 點擊下拉選單，選擇「<extern>」（外部控制器）。
  4. 點擊「Wizard」（精靈），選擇「Create a new controller」。
  5. 輸入名稱（如「avoid_obstacle」），選擇語言（如 Python）。
  6. Webots 會生成一個簡單的控制器檔案。
步驟 4：編輯控制器
  1. 在控制器檔案中輸入簡單的避障邏輯（範例 Python 程式碼）：

      • 感測器是否啟用。
      • 控制器是否正確綁定。
